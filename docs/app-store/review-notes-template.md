# App Review Notes Template

Use this in App Store Connect -> App Review Information -> Notes.

## Test account
- Not required in current build (no sign-in flow).

## 3-step reviewer flow
1. Open the app and add a Plan block from the timeline.
2. Tap the Plan block checkbox to auto-create and link a Done block.
3. Open Settings to verify Legal & Support information and Reset all data.

## Non-obvious features worth noting
- Day screen modes: switch between `Compare`, `Plan`, and `Done` to change the timeline view.
- Month view: open the calendar month screen to review daily completion percentages and jump into a specific day.
- Sharing/import: from Settings, share a day summary as text and import a shared summary into another day.
- Copy plan forward: from Settings, copy planned blocks from one date into another date.
- No account or backend is required in this build. All timeline data is stored locally on-device unless the user explicitly shares exported text.
