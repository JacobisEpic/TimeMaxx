# App Review Notes Template

Use this in App Store Connect -> App Review Information -> Notes.

## Test account
- Not required in current build (no sign-in flow).

## 3-step reviewer flow
1. Open the app and add a Plan block from the timeline.
2. Add a Done block and optionally link it to the Plan block.
3. Open Settings to verify Legal & Support links and Reset all data.

## Hardware / location requirements
- None.
